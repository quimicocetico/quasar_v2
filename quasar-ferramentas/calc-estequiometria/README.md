# Tutor Inteligente de Estequiometria

## Contexto

O Tutor Inteligente de Estequiometria é um aplicativo web que visa ajudar os estudantes de Química a resolver problemas de estequiometria de forma interativa e divertida.

Até agora, o aplicativo tem a seguinte estrutura:

* Uma página de entrada com inputs para a equação química, reagente e produto
* Um painel do tutor que exibe mensagens de ajuda e orientação
* Uma paleta de cores em tom azul escuro e um tema moderno glossmorphin
* Uma lógica para validar os inputs e balancear as equações químicas
* Um botão de calcular que exibe os resultados da equação balanceada

Os próximos passos incluem:

* Implementar a lógica para calcular a massa molar e a massa de produto
* Adicionar mais funcionalidades ao aplicativo, como um histórico de cálculos e opções de unidades

## Objetivos

* Criar um aplicativo web que seja fácil de usar e entender
* Fornececer uma ferramenta útil para os estudantes de Química que precisam resolver problemas de estequiometria
* Desenvolver uma aplicação que seja escalável e mantível
